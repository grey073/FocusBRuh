# -Focuz Bruh-

FocusGuard Pro is a web-based productivity and focus tool designed to help users maintain attention, track distractions, and take structured breaks. It uses real-time face detection to monitor focus and provides visual and auditory alerts for distractions.

---

## Features

- **Focus Timer:** Start predefined or custom focus sessions.
- **Break Timer:** Take scheduled breaks with a visual progress bar.
- **Distraction Tracking:** Counts face absence or excessive movement as distractions.
- **Tab Switch Monitoring:** Tracks when the user switches browser tabs during focus.
- **Streak System:** Rewards users for consecutive distraction-free sessions.
- **Audio Alerts:** Random alarms play when distractions are detected.
- **Responsive UI:** Clean, modern interface with smooth animations.

---
